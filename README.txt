This project contains the following:

  -template: This is the directory whose contents you copy to your project and
  customize. It provides you with a common-sense scss structure and
  configuration file, along with handy defaults and helpful hints.

  -core: This contains reusable components that you can load in throughout your
  projects from this central location. It serves as the repository for anything
  that can be used across projects, including css components, Sass mixins, and
  Sass functions.

Getting Started:

  -Make sure you've fulfilled the requirements listed below.
  -Copy the contents of the template directory to your project.
  -In your project, open config.rb.
  -Update the "Loading Core" path with where you've stored the core folder.
  -Run "compass watch" in your project directory and get started!

Requirements:

  To use this project, you'll need to install:

    -Ruby (http://www.ruby-lang.org/en/downloads/)
    -Sass (sudo gem install sass)
    -Compass (sudo gem install compass)
